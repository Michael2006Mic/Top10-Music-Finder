# Top10-Music-Finder
This is a project which uses Gemini AI to give you or search for the top 10 songs in the particular region you ask for.

The secrets.toml file in the file list has to be updated with your own API key (GEMINI API) . As this works on Gemini 2.5 Pro platform this is the requirement

HOW TO GET YOUR API KEY 
(In your web browser type gemini api key , click on the first link which pops up and follow the instructions on screen to get your api key , copy and paste the api key in the secrets.toml folder and run the program)

HOW TO RUN THE PROGRAM 
(Download both the app.py file and secrets.toml file on your computer and save it in a folder of your desired name , use the command [cd "Your file path"] and then use the command [python -m streamlit run app.py]

This will open a new tab in your web browser showing the final display
